# Miniblog

Um projeto de blog desenvolvido em React (estruturado via Create React App) utilizando Firebase para autenticação e gerenciamento de dados.

## 🚀 Tecnologias Utilizadas

* **React** (Biblioteca principal)
* **React Router** (Navegação/Roteamento)
* **Firebase** (Autenticação e Banco de Dados)
* **CSS** (Estilização)
